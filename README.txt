날씨비서

개발자 : 한혜진, 제서영, 차정은, 이시경

대한민국 어디서나 날씨를 보다 정확하게 확인하며 사용자가 원할 때 언제든 사용자가 일정을 편하게 계획하고 수행할 수 있도록 돕는 서비스

주요기능
- 맞춤형 개인화 서비스(날씨정보 제공, 아이템 추천, 오늘 활동 추천 등)
- 지역별(구/동/면) 맞춤 날씨 제공
- 친구와의 빠른 일정 공유
- 워치 맟춤 브리핑 알림 서비스
- 워치 맞춤 알림 서비스
- 타 캘린더와 연동시 일정을 한 번에 확인, 관리
